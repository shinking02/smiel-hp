# タスク完了時のチェックリスト

## 必須実行コマンド（順番重要）

### 1. 自動修正・フォーマット
```bash
npm run fix
```
- Prettierで自動フォーマット
- ESLintで自動修正可能な問題を解決

### 2. リント確認
```bash
npm run lint
```
- ESLintとPrettierを並列実行
- すべてのエラー・警告を確認

### 3. ビルド確認
```bash
npm run build
```
- TypeScriptコンパイルエラーがないことを確認
- 本番ビルドが成功することを確認

## エラーが発生した場合
1. リント・フォーマットエラー → `npm run fix` で自動修正を試す
2. TypeScriptエラー → 型の問題を手動で修正
3. ビルドエラー → インポート・設定の問題を修正

## 推奨事項
- コミット前に上記3コマンドを必ず実行
- CIが設定されている場合は、ローカルでの確認を徹底
- 大きな変更の場合は `npm run dev` で動作確認も実施

## 注意
- `npm run lint` が失敗した状態でコミットしない
- ビルドエラーがある状態でプッシュしない
